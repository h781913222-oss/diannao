> 版本：v1.0  
> 最后更新：2025年  
> 技术栈：Next.js + Tailwind CSS + Supabase + Vercel

---

## 一、项目概述

### 1.1 项目简介
一个面向普通用户与技术爱好者的综合导航聚合网站，整合软件下载、GitHub精选、在线工具、AI工具、硬件配置推荐、系统优化教程等内容。背靠实体电脑店，提供专业的硬件知识与装机配置推荐，是本站的核心差异化优势。

### 1.2 目标用户
- **普通用户**：需要软件下载、工具推荐、电脑入门知识的大众用户
- **技术爱好者**：关注GitHub开源项目、AI工具、系统优化的进阶用户
- **装机用户**：有购机、装机需求，需要配置推荐和报价咨询的用户

### 1.3 核心价值
- 帮用户筛选过的优质资源，节省查找时间
- 专业电脑店背景，硬件推荐真实可靠
- 持续更新，内容随时代保持新鲜

## 四、功能详细说明

### 4.1 全站搜索
- 位置：顶部导航栏 + 首页Hero区域大搜索框
- 搜索范围：工具名称、简介、标签、分类
- 搜索结果实时展示，无需跳转页面
- 支持中英文搜索

### 4.2 暗黑模式
- 右上角切换按钮（太阳/月亮图标）
- 用户选择记录在 localStorage，下次访问自动应用
- 跟随系统默认设置（prefers-color-scheme）

### 4.3 分类导航
六大主分类：
| 分类 | Slug | 图标 | 说明 |
|------|------|------|------|
| 软件下载 | software | 💾 | 常用软件、安装教程 |
| GitHub精选 | github | 🐙 | 优质开源项目推荐 |
| 在线工具 | tools | 🔧 | 图片压缩、格式转换等 |
| AI工具 | ai | 🤖 | AI写作、绘图、视频等 |
| 硬件推荐 | hardware | 🖥️ | 装机配置、选购指南 |
| 系统优化 | system | ⚡ | Windows技巧、驱动教程 |

### 4.4 工具卡片
每张卡片包含：
- 网站图标（favicon，自动抓取）
- 工具名称
- 一句话简介（不超过30字）
- 场景标签（2-3个，如：免费、AI、写作、开发者）
- 编辑推荐标记（🔥热门 / ⭐编辑推荐）
- 一键跳转按钮（新标签页打开）

### 4.5 场景标签系统
预设标签池，每个工具可打2-3个标签：

**免费相关**：免费、免注册、开源

**用户类型**：小白友好、开发者、设计师、办公人群

**功能类型**：AI、写作、图片、视频、文件、网络、编程、系统、硬件

**推荐程度**：编辑推荐、热门、新上线

### 4.6 编辑推荐标记
- 🔥 热门：访问量高、用户常用
- ⭐ 编辑推荐：经过你们亲测推荐
- 🆕 新上线：最近7天新增
- 以上标记在Supabase后台直接勾选

### 4.7 今日热榜
- 位置：首页显眼位置
- 内容：你们每周手动挑选5-10个热门工具
- 展示方式：横向滚动卡片或列表排行
- 在Supabase后台设置 is_hot=true 即可上榜

### 4.8 配置单生成器
- 位置：硬件推荐页 /pc-build
- 用户选择预算档位，展示对应配置推荐
- 预算档位：
  - 💰 3000以内（入门办公）
  - 💰 5000左右（家用主流）
  - 💰 8000左右（高性能）
  - 💰 10000以上（旗舰发烧）
- 每套配置包含：CPU、主板、内存、硬盘、显卡、机箱电源
- 底部有联系按钮，引导用户咨询或到店

### 4.9 工具投稿入口
- 位置：导航栏 + 页脚均有入口
- 页面：/submit
- 表单字段：
  - 网站名称（必填）
  - 网站链接（必填）
  - 网站简介（必填，限50字）
  - 推荐分类（下拉选择）
  - 推荐理由（选填）
  - 提交者联系方式（选填）
- 提交后发送邮件通知你们审核，审核通过后手动在Supabase上线

### 4.10 联系我们
- 页面：/contact 或页脚展示
- 展示信息：
  - 📧 邮箱：[邮箱占位符]
  - 💬 微信：[微信号占位符]
  - 🎵 抖音：[抖音号占位符]
- 可加一句话说明：有装机需求、合作投稿、问题反馈均可联系

## 五、数据库表结构（Supabase）

### 5.1 tools 表（工具/资源主表）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | uuid | 主键，自动生成 |
| name | text | 工具名称 |
| url | text | 跳转链接 |
| description | text | 一句话简介 |
| category | text | 分类slug（software/github/tools/ai/hardware/system） |
| tags | text[] | 场景标签数组 |
| icon_url | text | 图标地址（可自动抓取favicon） |
| is_featured | boolean | 是否编辑推荐 |
| is_hot | boolean | 是否热门/上热榜 |
| is_new | boolean | 是否新上线 |
| is_published | boolean | 是否已发布（审核开关） |
| sort_order | integer | 排序权重，数字越大越靠前 |
| created_at | timestamp | 创建时间 |
| updated_at | timestamp | 更新时间 |

### 5.2 submissions 表（用户投稿表）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | uuid | 主键 |
| name | text | 投稿网站名称 |
| url | text | 投稿链接 |
| description | text | 简介 |
| category | text | 推荐分类 |
| reason | text | 推荐理由 |
| contact | text | 提交者联系方式 |
| status | text | 状态：pending/approved/rejected |
| created_at | timestamp | 提交时间 |

### 5.3 pc_builds 表（配置单表）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | uuid | 主键 |
| budget_label | text | 预算档位名称 |
| budget_min | integer | 最低预算（元） |
| budget_max | integer | 最高预算（元） |
| cpu | text | CPU型号 |
| motherboard | text | 主板 |
| memory | text | 内存 |
| storage | text | 硬盘 |
| gpu | text | 显卡 |
| case_psu | text | 机箱+电源 |
| total_price | integer | 参考总价 |
| use_case | text | 适用场景说明 |
| updated_at | timestamp | 最后更新时间 |

## 六、页面设计规范

### 6.1 整体风格
- 简洁现代，内容优先
- 卡片式布局，信息密度适中
- 支持亮色/暗色双主题

### 6.2 颜色系统
- 主色：蓝色系（科技感）
- 亮色背景：#ffffff / #f8fafc
- 暗色背景：#0f172a / #1e293b
- 强调色：根据最终品牌色确定

### 6.3 响应式断点
- 手机：1列卡片
- 平板：2列卡片
- 桌面：3-4列卡片

### 6.4 卡片尺寸
- 固定高度卡片，信息整齐
- 图标 32x32px
- 名称加粗，简介灰色小字
- 标签用圆角小胶囊样式

---

## 七、SEO要求
- 每个页面有独立的 title 和 meta description
- 分类页标题格式：「{分类名} - 【网站名】」
- 图片均有 alt 属性
- 使用语义化HTML标签
- 生成 sitemap.xml
- 页面加载速度优先，图片懒加载
