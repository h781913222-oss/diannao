## 三、Hero 背景规范

### Aurora 效果

| 参数 | 值 | 说明 |
|------|-----|------|
| 颜色1（深蓝） | `vec3(0.05, 0.1, 0.2)` | 基础氛围 |
| 颜色2（紫） | `vec3(0.4, 0.0, 0.8)` | 主色调 |
| 颜色3（青） | `vec3(0.0, 0.8, 0.8)` | 点缀色 |
| 混合强度 | `n1 * 0.5 + 0.5` | 适度波动 |
| 聚光灯强度 | `0.15` | 鼠标交互反馈 |
| 最终混合 | `0.6` | 保持可感知但不抢主体 |

### 使用规则

- Aurora 为氛围背景，不抢主体
- 保持沉浸感，不喧宾夺主
- 颜色克制，不使用高饱和度色彩

## 四、ParticleScene 规范

### 粒子参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 粒子数量 | 1500 | 足够氛围感但不过密 |
| 粒子颜色 | 青色 `[0.0, 1.0, 1.0]` | 与 tertiary 呼应 |
| 粒子透明度 | 0.6 | 微弱但可感知 |
| 粒子大小 | 0.015 | 保持微弱 |

### 使用规则

- 粒子作为氛围点缀
- 不破坏沉浸感
- 保持微弱存在感

## 五、Glass Panel 规范

### 标准样式

| 参数 | 值 | 说明 |
|------|-----|------|
| 背景 | `rgba(255, 255, 255, 0.03)` | 轻量玻璃效果 |
| 模糊 | `backdrop-blur(12px)` | 适度模糊 |
| 边框 | `rgba(255, 255, 255, 0.08)` | 保持层次 |
| hover 边框 | `rgba(255, 255, 255, 0.15)` | 交互反馈 |
| 辉光 | `shadow-[0_0_100px_rgba(76,215,246,0.1)]` | 层次感 |

### 使用规则

- 轻量玻璃效果，保持层次
- 避免过度发光
- hover 效果克制

## 六、Typography 规范

### 字体栈

| 用途 | 字体 | 说明 |
|------|------|------|
| 标题 | Geist | 现代无衬线 |
| 正文 | Inter | 可读性优先 |
| 代码/标签 | Geist Mono | 技术感 |

### 标题渐变

| 参数 | 值 | 说明 |
|------|-----|------|
| 样式 | `bg-gradient-to-b from-white to-white/40` | 从白色到半透明白色 |
| 用途 | Hero 标题 | 提升层次感 |

### 使用规则

- 标题为视觉焦点
- 渐变仅用于提升层次
- 保持克制，不使用花哨效果

## 七、Spacing 规范

### 页面间距

| 场景 | 值 | 说明 |
|------|-----|------|
| Hero 留白 | `pt-24 pb-16` | 上下留白 |
| Section 间距 | `py-24` | 区块间隔 |
| 卡片间距 | `gap-6` | 网格间距 |
| 内边距 | `p-6` / `p-8` | 卡片内边距 |

### 使用规则

- 强调留白
- 保持高级 SaaS 节奏
- 避免内容拥挤
